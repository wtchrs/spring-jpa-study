# SpringMockShop

## Stack
- Spring 5
- Spring Boot 2.7
- Thymeleaf 3
- JPA
- H2 Database

## Getting started

Install JDK 17, or change `sourceCompatibility` setting in `build.gradle` before build.

#### Build:

```shell
./gradlew build
```

#### Run:

```shell
./gradlew bootRun
```

Or run created jar archive directly:

```shell
java -jar ./build/libs/SpringMockShop-0.0.1-SNAPSHOT.jar
```
